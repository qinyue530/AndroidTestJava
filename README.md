# AndroidTestJava

学习安卓开发的练习项目。

## 项目简介

本项目是一个 Android 学习项目，用于练习和掌握 Android 开发的基础知识和技能。

## 技术栈

- **语言**: Java / Kotlin
- **构建工具**: Gradle
- **IDE**: Android Studio

## 项目结构

```
AndroidTestJava/
├── app/                    # 应用主模块
├── .gradle/               # Gradle 缓存
├── .idea/                 # IDE 配置文件
├── build.gradle.kts       # 项目级构建脚本
├── gradle.properties      # Gradle 属性配置
├── settings.gradle.kts    # 项目设置
└── gradle/                # Gradle wrapper
```

## 开发环境要求

- Android Studio Arctic Fox 或更高版本
- JDK 11 或更高版本
- Android SDK

## 运行项目

1. 克隆仓库
   ```bash
   git clone https://github.com/qinyue530/AndroidTestJava.git
   ```

2. 使用 Android Studio 打开项目

3. 同步 Gradle 并运行
